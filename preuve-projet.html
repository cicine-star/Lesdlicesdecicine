Description du projet

Site e-commerce et panneau d'administration sur mesure pour une activité de pâtisserie artisanale maison. Le site permet aux clients de passer des commandes et des devis en ligne, et à la propriétaire de gérer ses produits, commandes, saveurs et stocks depuis un espace admin sécurisé, avec synchronisation en temps réel.

Fonctionnalités réalisées (au 23 juin 2026)
  • Page commande interactive avec catalogue en accordéons par catégorie, sélecteur de saveurs automatique pour les gâteaux, calcul du total en temps réel, envoi par WhatsApp, calendrier avec dates bloquées
  • Page « Nos produits » avec chargement dynamique depuis la base de données (mise à jour automatique)
  • Page devis, page avis clients, page à propos, CGV
  • Espace administration sécurisé (9 onglets) : Commandes, Stocks, Planning, Tâches, Statistiques, Clients, Produits, Saveurs, Paramètres
  • Gestion des saveurs par produit avec indicateur visuel vert/rouge (disponible / indisponible)
  • Base de données Supabase : produits, commandes, clients, saveurs, stocks, avis, dates bloquées
Technologies utilisées
  • HTML · CSS · JavaScript (site statique, sans dépendance externe)
  • Supabase (base de données temps réel hébergée)
  • GitHub Pages (hébergement gratuit et versionné)
  • WhatsApp Business API (transmission des commandes)
